# PIXEL FIGHTER 🥊

PIXEL FIGHTER is a fast-paced, retro-style multiplayer fighting game built with **Three.js** and **WebSockets**. Battle your friends in real-time using unique characters with special abilities and ultimate skills.

## 🎮 Features

- **Real-time Multiplayer**: Powered by WebSockets for low-latency combat.
- **Game Modes**: Choose between **1v1** duels or chaotic **Deathmatch** battles.
- **Unique Characters**:
    - 🗡️ **Shadow Assassin**: Fast and elusive.
    - ❄️ **Frost Mage**: Controls the battlefield with ice.
    - 🌋 **Flame Berserker**: High damage and raw power.
    - 🛡️ **Tech Guardian**: Advanced defense and technology.
    - 🏹 **Storm Archer**: Precise ranged attacks.
- **Cross-Platform**:
    - **Desktop**: Full keyboard support.
    - **Mobile**: Custom on-screen D-Pad and action buttons.
- **WebGL UI**: Custom-built menus, character selection, and an interactive numpad for room codes.
- **Dynamic Audio**: Retro-inspired soundtrack with multiple tracks and sound effects.

## 🕹️ How to Play

### Getting Started
1. **Host**: Click "Host Game" to create a room. Share the 4-digit code with your friends.
2. **Join**: Click "Enter Code", use the numpad to type the code, and click "Join Room".
3. **Select**: Choose your character and enter your name.

### Controls
| Action | Desktop | Mobile |
|--------|---------|--------|
| **Move** | Arrow Keys | D-Pad |
| **Attack** | Spacebar | 'A' Button |
| **Ultimate** | 'E' Key | 'U' Button |

## 🛠️ Development

**Prerequisites:** Node.js

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run the app**:
   ```bash
   npm run dev
   ```
